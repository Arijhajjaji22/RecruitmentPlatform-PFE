## Description
Web platform for managing recruitment processes, including internships and candidate applications.  
Integrated DevOps pipeline ensures automated testing and deployment.
## Demo Video
Watch the demo video here: [Demo Video]( https://drive.google.com/file/d/1x6rwel4OfuMKxHB58Nx5pQYSKrzvLA1E/view?usp=sharing )

## Features
- **Employee Management**: Add, edit, and manage employee data.
- **Internship Offers Management**: Create and track available internship positions.
- **Candidate Applications Management**: Track and evaluate applications.
- **Interview Management**: Schedule and manage interviews for candidates.
- **Tests Management**: Create and manage candidate assessment tests.

## Tech Stack
- Backend: .NET 8, C#
- Frontend: Razor, HTML5, CSS3, JavaScript
- Database: MySQL
- DevOps: Jenkins, Docker, XUnit, Prometheus, Grafana
- Version Control: Git
